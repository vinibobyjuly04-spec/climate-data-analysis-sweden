# Climate Data Analysis – Växjö, Sweden

## 📊 Project Overview
This project analyzes 30 years of temperature and precipitation data from Växjö, Sweden using statistical modeling and time series forecasting techniques.

## 🔧 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- R (Statistical Modeling, Copula Analysis)
- SQL (Data Handling)
- Excel

## 📈 Key Techniques Used
- Time Series Analysis (ARIMA, SARIMA)
- Copula Modeling (Clayton & Gumbel)
- Extreme Value Theory (EVT)
- Data Transformation (Box-Cox, Log)
- Kendall’s Tau Correlation

## 🔍 Key Insights
- Identified seasonal temperature and precipitation patterns
- Built predictive models for climate forecasting
- Analyzed extreme temperature events using EVT
- Developed mixture copula model for dependency analysis
